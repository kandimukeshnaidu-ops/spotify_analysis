# Spotify Music Analysis (EDA)

## Overview
This project performs exploratory data analysis on a Spotify music dataset to understand how audio features, genres, and song characteristics relate to popularity.

## Dataset
The dataset contains over 100,000 tracks with audio features such as danceability, energy, loudness, valence, and genre information.

## Objectives
- Analyze the distribution of song popularity
- Compare audio characteristics across genres
- Identify audio features associated with popular songs
- Compare popular and unpopular tracks

## Key Insights
- Most songs on Spotify have low popularity, with only a small fraction achieving high popularity.
- Danceability varies significantly across genres, with rhythm-focused genres showing higher values.
- Popular songs tend to be more energetic and louder, while instrumental tracks are generally less popular.
- No single audio feature guarantees popularity, indicating that success depends on multiple factors.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Limitations
- Lack of release year data prevents trend analysis over time.
- External factors influencing popularity are not included.

## Conclusion
This analysis highlights how audio characteristics and genres influence song popularity while emphasizing that popularity is multifactorial.
